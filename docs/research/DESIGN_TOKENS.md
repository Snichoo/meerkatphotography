# Design Tokens

## Colors
| Token        | Value            | Use |
|--------------|------------------|-----|
| cream        | `#EFE4D8` rgb(239,228,216) | page background, text on navy |
| navy / ink   | `#2B2D3F` rgb(43,45,63)    | text, navy bands, footer, primary button |
| gold         | `#C09166` rgb(192,145,102) | accent text/links/borders |
| orange       | `#DD6731` rgb(221,103,49)  | small accent (chips) |
| white        | `#FFFFFF`        | button text |
| overlay      | `rgba(0,0,0,0.74)` | hero/nav overlay |

## Typography  Poppins (sans-serif)
- Body: 16px / weight 400 / color #2B2D3F
- H1: ~46px / weight 400 / line-height ~60px
- H2: ~46px / weight 600 / UPPERCASE, letter-spacing normal
- Weights used: 300, 400, 500, 600

## Buttons
- **Primary pill** (nav "Get a quote"): bg #2B2D3F, color #fff, border-radius 90px, fw 300, ~17px.
- **Outline** ("GET A QUOTE"): transparent, 4px solid (#2B2D3F on cream / #EFE4D8 on navy),
  border-radius 20px, padding 4px 15px 2px, fw 300, ~22px.

## Motifs
- Large navy semicircle / arch (border-radius 50% top) transitions between cream & navy bands.
- Dotted vertical divider under service titles.
- Generous vertical whitespace; centered headings.
